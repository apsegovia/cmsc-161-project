Specs

Project Requirements
The project must demonstrate, in an integrated manner, the following topics:

1. 2D Geometry (primitives, buffers, basic interaction)
2. 3D Objects & Viewing (scene construction, camera/view space)
3. Transformations & Homogeneous Coordinates (translation, rotation, scaling, composition)
4. Viewing & Projection (perspective/orthographic projection, viewing volumes)
5. Lighting and Shading (Phong or equivalent model)
6. Texture Mapping (image-based surface detail)
7. Shadows (e.g., shadow mapping or a simplified approximation)

D. Project Description
We (each one of us) will develop a WebGL-based interactive scene that:

1. Models a coherent environment
   (e.g., a room, outdoor scene, mechanical system, mini-game, or simulation)
2. Contains multiple 3D objects
   ► At least one hierarchical or articulated object
   ► Use of reusable geometry or instancing
3. Supports user interaction
   ► Camera movement (e.g., orbit, pan, zoom)
   ► At least one object-level interaction (e.g., rotation, animation trigger)
4. Implements realistic rendering features
   ► Lighting and shading (Phong or similar)
   ► Texture mapping on at least two surfaces
   ► Shadow rendering (shadow mapping or simplified technique)
5. Demonstrates transformation pipelines
   ► Proper use of modeling, viewing, and projection transformations
   ► Clear distinction between coordinate spaces

E. Minimum Technical Requirements
Our respective individual projects must include:
● WebGL program written in HTML + JavaScript + GLSL shaders
● At least:
 5 distinct objects in the scene
 2 textured surfaces
 1 movable light source or camera
● Use of:
 Vertex and fragment shaders
 Buffer objects for geometry
 Transformation matrices (model/view/projection)
● Implementation of:
 Perspective projection
 Basic shadowing technique
